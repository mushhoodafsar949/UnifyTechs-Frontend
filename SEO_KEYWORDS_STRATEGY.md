## 📌 HOW TO USE THIS DOCUMENT

Keywords are organized into **7 strategic tiers** based on:
- **Search Intent**: Informational (learning) → Commercial (comparing) → Transactional (buying)
- **Competition Level**: Low 🟢 · Medium 🟡 · High 🔴
- **Priority**: Must-rank 🔥 · Important ⭐ · Supporting 📌

**Golden Rule**: Target 1 primary keyword + 2–3 secondary + 5–8 long-tails per page.
High-competition terms need dedicated landing pages + blog authority; low-competition terms convert faster.

## ⚙️ TECHNICAL SEO CHECKLIST

These technical factors amplify the impact of all your keyword targeting:

**Page Speed** — Aim for a Google PageSpeed score of 90+ on both mobile and desktop. Use Pexels CDN params (`?auto=compress&cs=tinysrgb&w=1200`) on all images as already specified in the Cursor prompt.

**Core Web Vitals** — LCP under 2.5s, FID under 100ms, CLS under 0.1. These directly affect ranking.

**Mobile First** — Google indexes mobile version first. The React site must be fully responsive.

**URL Structure** — Keep URLs clean and keyword-rich: `/services/claim-automation`, `/blog/reduce-claim-denials`, `/compare/waystar-alternative`.

**Internal Linking** — Every blog post should link to at least 2 service pages. Every service page should link to the demo page. This flows "link juice" from informational content to commercial pages.

**Canonical Tags** — Prevent duplicate content penalties on any pages that might be similar.

**Sitemap** — Submit XML sitemap to Google Search Console on launch day.

**Robots.txt** — Allow all crawling. Block only admin and staging URLs.

**SSL/HTTPS** — Non-negotiable. Also required for HIPAA-adjacent trust.
